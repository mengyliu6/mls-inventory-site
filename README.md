# 公司库存管理网站

这是一个公司自用库存管理网站的第一版原型，直接打开 `index.html` 即可使用。

## 已包含功能

- 房间1、房间2、房间3、房间4
- 仓库房间与大货架图示页面
- 大货架拖拽布局编辑
- 每个大货架自动分为 8 个货位，例如 `B1-1` 到 `B1-8`
- 大货架详情弹窗，可查看每个货位中的多款产品
- 在货位产品上直接出库或编辑
- 产品库存表格管理
- 出入库登记，默认当前时间，可手动修改时间
- 按人员分配大货架编辑权限，不需要审批流程
- 最近出入库记录
- 销量排行、库存占比、每日出入库趋势图
- 浏览器本地保存数据
- JSON 数据导出

## 当前数据模型

- 房间：固定为房间1、房间2、房间3、房间4
- 大货架：属于某个房间，带编号和图示位置
- 货位：每个大货架上下四层、左右两列，共 8 个空间
- 产品：车型、型号、图片、系统版本、储存配置、货位、备注、库存
- 出入库记录：产品、类型、数量、时间、操作人、备注
- 人员权限：管理员、仓库编辑、只读查看

## 后续建议

第一版是前端原型，适合确认流程和页面结构。确认后建议升级为正式系统：

- 后端接口与数据库
- 登录和真实用户权限
- 图片上传
- Excel 导入导出
- 出入库记录不可篡改审计
- 多人同时编辑冲突处理

## 需要确认的信息

- 房间布局：房间1、2、3、4 中分别有几个大货架，实际大致位置如何
- 大货架编号：例如 `B1`、`B2`、`C1`，系统会自动生成 `B1-1` 到 `B1-8`
- 产品字段：当前保留车型、型号、图片、系统版本、储存配置、货位、备注、库存
- 出入库字段：当前保留产品、数量、时间、操作人、备注
- 权限规则：人员是否只能编辑指定大货架
- 库存预警：低库存阈值是按产品统一设置，还是每个产品单独设置
- 数据导入：现有 Excel 是否需要一次性导入到系统
- 部署方式：只在公司内网使用，还是需要公网远程访问
