# 公司库存管理网站

这是一个公司自用库存管理网站的第一版原型，直接打开 `index.html` 即可使用。

## 已包含功能

- 仓库房间与货架图示页面
- 货架拖拽布局编辑
- 产品库存表格管理
- 出入库登记，默认当前时间，可手动修改时间
- 按人员分配货架编辑权限
- 最近出入库记录
- 销量排行、库存占比、每日出入库趋势图
- 浏览器本地保存数据
- JSON 数据导出

## 当前数据模型

- 房间：例如 A 区房间、B 区房间、C 区房间
- 货架：属于某个房间，带货架号和图示位置
- 产品：车型、型号、图片、安卓版本、储存配置、货架、备注、库存
- 出入库记录：产品、类型、数量、时间、操作人、备注
- 人员权限：管理员、仓库编辑、只读查看

## 后续建议

第一版是前端原型，适合确认流程和页面结构。确认后建议升级为正式系统：

- 后端接口与数据库
- 登录和真实用户权限
- 图片上传
- Excel 导入导出
- 出入库记录不可篡改审计
- 多人同时编辑冲突处理

## 需要确认的信息

- 房间清单：每个房间的名称、区域编号、实际尺寸或大致平面布局
- 货架规则：货架号命名方式，例如 `A-2`、`B-1-3`、`C2-6`
- 产品字段：是否还需要供应商、采购价、销售价、批次号、条码、序列号
- 出入库字段：是否需要客户、订单号、平台、快递单号、审核人
- 权限规则：人员是否只能编辑指定货架，是否需要审批出库
- 库存预警：低库存阈值是按产品统一设置，还是每个产品单独设置
- 数据导入：现有 Excel 是否需要一次性导入到系统
- 部署方式：只在公司内网使用，还是需要公网远程访问
